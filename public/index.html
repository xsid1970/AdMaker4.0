📘 초보자 가이드 (메뉴얼)

🚀 5단계 퀵 스타트 (완전 초보용)

처음이신가요? 이 순서대로만 따라하면 광고 하나가 뚝딱 완성됩니다!

  1. 1. 템플릿 선택: 오른쪽 '🎨 템플릿 열기'를 눌러 마음에 드는 디자인을 고르세요.
  2. 2. 내용 수정: 화면 속 글자를 클릭하고 오른쪽 '내용 입력' 칸에 실제 문구를 적으세요.
  3. 3. 사진 교체: 사진을 바꾸려면 선택 후 '파일 선택'을 누르거나 '자르기(✂️)'로 영역을 잡으세요.
  4. 4. 꾸미기: '✨ 스티커' 메뉴에서 원하는 뱃지를 더블 클릭하여 배치하세요.
  5. 5. 저장 및 출력: 다 했다면 '💾 내역 저장'을 누르고 'JPG 다운로드'를 클릭하세요!

🏢 자동 완성 & 정보 입력 (꿀팁!)

상호명을 입력하면 기존에 작업했던 데이터를 자동으로 찾아줍니다.

  • 이름이나 연락처가 자동으로 채워지면 노란색 반짝임 효과가 나타납니다.
  • 목록에서 원하는 업체를 클릭하면 즉시 모든 정보가 입력되어 시간이 절약됩니다.

🖼️ 이미지 & 사진 정밀 도구

보유하신 사진을 올리고 원하는 부분만 깔끔하게 잘라낼 수 있습니다.

  • 사진 버튼: 내 PC의 이미지를 불러와서 즉시 교체합니다.
  • 자르기(✂️): 이미지 선택 후 '자르기' 버튼을 눌러 마우스로 영역을 조절하세요.
  • QR코드: 홍보용 주소를 입력하면 스마트폰으로 찍을 수 있는 코드가 자동 생성됩니다.

🔓 고급 편집 모드 (자유로운 이동)

기본 수정 외에 위치를 옮기거나 크기를 자유롭게 조절하고 싶을 때 사용하세요.

  • 위쪽의 '🔓 고급 편집 잠금' 버튼을 눌러 해제하면 모든 요소를 마우스로 끌어서 옮길 수 있습니다.
  • 레이어 조절: 여러 요소가 겹칠 때 '앞으로/뒤로' 버튼으로 순서를 바꿉니다.
  • 단축키: 복제 Alt + D / 취소 Ctrl + Z / 복사 Ctrl + C

💾 작업 내역 관리

작업한 내용은 언제든 다시 꺼내서 수정할 수 있습니다.

  • 저장 필수: 작업 후 '내역 저장'을 해야 나중에 다시 불러올 수 있습니다.
  • 불러오기: 왼쪽 업체 목록에서 업체 이름을 누르면 이전 작업이 즉시 로드됩니다.
