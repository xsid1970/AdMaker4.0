# AdMaker 원클릭 배포 워크플로우

이 워크플로우는 `// turbo-all` 설정이 되어 있어, 안티그래비티가 배포 과정을 자동으로 진행할 때 사용자의 승인 과정을 최소화합니다.

## 실행 방법
1. 터미널(PowerShell)에서 `deploy.ps1`을 실행합니다. 
   - 커밋 메시지를 지정하려면 인자로 전달할 수 있습니다. (`./deploy.ps1 "기능 개선 설명"`)
2. 안티그래비티가 `SafeToAutoRun` 설정을 통해 배포를 진행합니다.

## 배포 스크립트 실행
// turbo
```powershell
.\deploy.ps1
```

## 완료 확인
- 배포가 완료되면 브라우저에서 `http://13.125.184.180`에 접속하여 변경 사항을 확인합니다.
